# Image to Content Generator using Generative AI

## 📌 Project Overview
This project uses Generative AI to analyze images and generate meaningful textual content automatically. Users can upload an image,
and the model extracts visual information to create descriptions, captions, or content based on the image.

## 🚀 Features
- Upload image files
- AI-powered image understanding
- Automatic content generation
- Fast and accurate results
- Supports multiple image formats
